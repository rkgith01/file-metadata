File Metadata Microservice

Usage:

Please Upload a File ...

How it works

  • Upload Trigger: When a user selects a file and clicks "Upload", the form data is sent to the server via a POST request to the /api/fileanalyse endpoint.
  • Server Interaction: The server processes the uploaded file and returns metadata including its name, type, and size as a JSON response.
  • Sugestion: Dynamic Display: The received metadata is dynamically displayed on the webpage in the designated section, providing instant feedback to the user without needing a page refresh.
  • JSON Preview: On clicking the upload button user are redirected to view the file in JSON format.

How the response looks 

        
          {
            "name": "logo.png",
            "type": "image/png",
            "size": 30588
          }
